# CPP Modules

C++ modules developed as part of the 42 School curriculum.  
This repository contains all C++ projects from Module 00 to Module 08, focused on Object-Oriented Programming, memory management, templates, and advanced C++98 features.

The goal of these modules is to learn C++ starting from C, and progressively understand OOP, polymorphism, inheritance, exceptions, templates, and STL. :contentReference[oaicite:0]{index=0}

## Modules

- Module 00 — Namespaces, classes, member functions, std::iostream, static, const
- Module 01 — Memory allocation, references, pointers to members
- Module 02 — Ad-hoc polymorphism, operator overloading, canonical form
- Module 03 — Inheritance
- Module 04 — Subtype polymorphism, abstract classes, interfaces
- Module 05 — Exceptions, try/catch
- Module 06 — C++ casts
- Module 07 — Templates
- Module 08 — STL containers, iterators, algorithms
- Module 09 — STL

Each module contains multiple exercises designed to reinforce C++ fundamentals and object-oriented design. :contentReference[oaicite:1]{index=1}

## Features

- C++98 standard only
- Object-oriented programming
- Operator overloading
- Canonical class form
- Inheritance & polymorphism
- Exception handling
- Templates
- STL containers
- Memory management
- Strict compilation flags
- STL

## Build

Each exercise can be compiled with:

```bash
make
```

Compilation flags:

```bash
-c -Wall -Wextra -Werror -std=c++98
```

## Run

Example

```bash
cd cpp00/ex00
make
./program
```
